# 绘制线

### 简介

线是在地图上的基本元素，由一组经纬度坐标点串组合而成，可用来表示驾车路线、步行轨迹等场景。

腾讯地图SDK对线提供了各种样式的属性配置接口，支持绘制虚线、分段线、纹理线。下面将分7小节来分别介绍线的内容：

1. 常用属性解释
2. 绘制一条简单的线
3. 动态改变线的属性
4. 在线上显示一串文字
5. 线的擦除
6. 线的动画
7. 移除线
